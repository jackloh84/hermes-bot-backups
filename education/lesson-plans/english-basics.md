# English Basics — Lesson Plan

**Level:** Beginner
**Age:** Child
**Duration:** 15-20 min

## Objectives
- Learn 5 new English words
- Practice simple sentence structure
- Build confidence speaking

## Vocabulary Set
1. Apple 🍎
2. Book 📚
3. Cat 🐱
4. Dog 🐶
5. Elephant 🐘

## Activities
1. **Flashcard review** — show picture, say word
2. **Fill in the blank** — "I see a ___"
3. **Drawing time** — draw the animal you learned

## Assessment
- Can the child say each word independently?
- Can they use it in a simple sentence?
